# YouTube Downloader (Python + Tkinter)

Aplicație GUI realizată în Python pentru descărcarea videoclipurilor YouTube folosind yt-dlp.

## Funcționalități
- Introducere link YouTube
- Alegere folder de salvare
- Validare link
- Mesaj de succes / eroare

## Tehnologii utilizate
- Python
- Tkinter
- yt-dlp

## Rulare aplicație

1. Instalează dependențele:
   pip install yt-dlp

2. Rulează aplicația:
   python youtube_downloader.py
